# AI開発成功事例データベース — プロジェクト指示

## まず読む
- **`PROJECT_GOAL.md`** — このプロジェクトの単一の真実の源(ゴール・収益モデル・確定した設計判断)。作業前に必ず参照する。
- `docs/DATA_MODEL.md` — 事例(Case)の確定スキーマ。
- `docs/COMPETITIVE_ANALYSIS.md` — 競合分析と読者調査。
- `content/types.ts` — スキーマのコード定義。`content/cases/` に事例データ。

## 技術スタック
Next.js (App Router) + TypeScript + Tailwind CSS。Vercelデプロイ。日英の両言語対応(`/ja` `/en` のパス分離 + hreflang)。
